## Installation

1. Clone repository

```bash
git clone https://github.com/arulGo18/Tracking-Package.git
```

2. Import database

```sql
database/tracking_packet.sql
```

3. Configure database

```php
config/database.example.php
```

Rename to:

```php
config/database.php
```

4. Run using XAMPP

```
http://localhost/tracking-packet
```
